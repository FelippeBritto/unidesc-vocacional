## Install

```
$ git clone https://github.com/gepittes/unidesc-vocacional
$ cd unidesc-vocacional
$ composer install
```

## Usage

```
$ cp .env.example .env
$ php artisan key:generate
```

Configure your database in **.env**

```
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=your_db
DB_USERNAME=root
DB_PASSWORD=123456
```

```
$ php artisan migrate --seed
```

## Run app

```
$ php artisan serve
```

Check app in http://localhost:8000/
